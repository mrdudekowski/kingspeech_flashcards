Grammar Categories – Icon Mapping Preview

Предпросмотр иконок для грамматических категорий из A1/grammar-words. Это только визуальное согласование; логика и данные берутся из плана.

Main category – верхний уровень (`category` в JSON: pronouns, prepositions, ...).

Subcategory – уточнение (`subcategory` в JSON: personalPronouns, objectPronouns, ...). В UI флешкарточек планируется использовать именно подкатегорию, если она есть, иначе – основную категорию.
