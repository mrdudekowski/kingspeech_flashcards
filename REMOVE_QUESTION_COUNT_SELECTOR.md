## 📊 ЛОГИКА РАБОТЫ

### Теперь:
- Квиз **всегда** использует **все** доступные слова из коллекции
- Пользователь **не может** выбрать количество вопросов
- Количество вопросов = количеству слов в коллекции

### Примеры:
```
Grammar Words (74 слова) → Квиз из 74 вопросов
Food & Drink (45 слов) → Квиз из 45 вопросов
Travelling (28 слов) → Квиз из 28 вопросов
```

## 📱 АДАПТАЦИЯ

Информационный блок использует правильное склонение:
```typescript
{words.length === 1 ? 'вопрос' : words.length < 5 ? 'вопроса' : 'вопросов'}
```

**Примеры:**
- 1 слово → "1 вопрос"
- 2 слова → "2 вопроса"
- 5 слов → "5 вопросов"
- 74 слова → "74 вопроса"
